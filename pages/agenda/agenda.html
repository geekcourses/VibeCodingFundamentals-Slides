Module 1: Introduction to Vibe Coding

Course Intro

Introduction to Vibe Coding. Basic Concepts and Principles of Generative Large Language Models (LLMs)

  1. What is Vibe Coding?
  2. History and popularization (Andrej Karpathy, 2025).
  3. Differences between traditional programming, "AI-assisted development" and Vibe Coding.
  4. LLMs: How they work, how they are trained and what is important to know to work with them effectively.

Ecosystem of Tools. Introduction to VSCode and GitHub Copilot

  1. Overview of modern AI tools (Replit, Bolt, Windsurf, ChatGPT, GitHub Copilot, Cursor, Claude Code, etc.)
  2. Discussion of their strengths and weaknesses
  3. How to choose the right AI tool for a specific task?
  4. Installation, basic interface, basic settings.

Vibe Coding Workflow. Demonstration

  1. Idea → prompt → code → review → version → test → deploy
  2. Moving "from idea to code with prompts only"
  3. Example of creating a simple Web application with Bolt.

Module 2: Prompt Engineering and Effective Communication with AI

Prompt Basics. Contextualization Techniques.

  1. Structuring clear and effective prompts.
  2. How to give AI the necessary context to get an accurate result (role playing, specifying response format, including example data).

Prompt Optimization Techniques

  1. Chain of thought, few-shot learning, iterative improvement, etc.

Creating Prompt Libraries and Practical Tips.

  1. Organization and reuse of effective prompts
  2. How to deal with common problems like "hallucinations" and inaccuracies.

Practice

  1. Creating a quick application prototype in various chat-based interfaces (ChatGPT/Claude/Gemini)

Module 3: Project Management with Git and AI

Basic Principles. Git and GitHub Integration in VSCode, AI Integration.

  1. Why version control is important.
  2. Using Git and GitHub for version control and project management.
  3. How to use built-in VSCode Git functions for version control directly from the editor.
  4. How to use AI to automatically generate commit messages and manage repositories.
